## Android

En android s'ha implementat varies funcionalitats, entre les quals, podem trobar:
- Registrarse
- Iniciar sessió
- FAQs
- Denunciar un abús
- Ranking global de jugadors
- Tenda
- Perfil
- Foro
- Tancas sessió
